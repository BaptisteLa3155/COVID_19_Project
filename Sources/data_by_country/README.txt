These CSV files are containing data about OECD member countries. 

The data is concerning four features: 
- The proportion of overweight people in percentage 
- The proportion of people older than 65 years in percentage
- The number of hospital beds for 1000 habitants 
- The number of healthcare workers for 1000
- The number of tests per confirmed cases 

The top four files come from the website ‘stats.oecd.org’ whereas the last one 
is available on github https://github.com/owid/covid-19-data/tree/master/public/data/.
